# Titanic Analysis (Assignment 2)

This folder contains the code for the Titanic dataset assignment.

## Structure
- `data/`: Put `train.csv` and `test.csv` here.
- `scripts/`: Python scripts for cleaning and features.
  - `data_cleaning.py`: handles missing ages and embarked ports.
  - `feature_engineering.py`: creates family size and encodes gender.
  - `feature_selection.py`: removes columns like ID and Name.
- `notebooks/`: Contains the `Titanic_Feature_Engineering.ipynb` walkthrough.
- `requirements.txt`: libraries needed.

## How to use
Run the scripts in order or open the notebook in `notebooks/`.
