# Football Results Analysis (Assignment 1)

This project analyzes historical international football results to discover trends in scoring and match outcomes.

## Structure
- `data/`:
  - `results.csv`: The raw dataset of international matches.
  - `results_analyzed.csv`: The dataset after processing and analysis.
- `scripts/`:
  - `football_analysis.py`: Python script that performs the full analysis and saves results.
- `notebooks/`:
  - `Football_Analysis.ipynb`: Interactive walkthrough of the findings.
- `assignment1_football_analysis.md`: Summary report of the results.

## How to run
Run the analysis script:
```bash
python scripts/football_analysis.py
```
Or view the interactive notebook in the `notebooks/` folder.
